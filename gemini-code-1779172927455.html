About The Event

沉浸於蔚藍的慢活節奏

日常太快，所以我們需要一處能容納慵懶的庇護所。Léger Café 的創立初衷，是將海洋的寬廣與天空的純淨帶入都市角落。

本次開幕活動不僅是新店宣告，更是一場感官放鬆之旅。現場將提供特別研發的夏日主題限定餐點，並由資深烘豆師與咖啡師現場分享極致風味的淬煉過程。
